== Description ==

Manage Your eCommerce Finances with Your Lili Account

Your Lili business checking account gives you everything you need to stay on top of your business finances—without leaving your platform. Instantly track your cash flow, automate expenses, and make sure you’re set for tax season, all in one place.
✅ Real-Time Financial Overview – View your latest transactions, checking account balance, and savings balance right inside your dashboard.
✅ Faster Payouts & Cash Flow Management – Move money quickly with express ACH, wire transfers, and fee-free overdraft*.
✅ Built-In Accounting** – Automate expense tracking, bill pay, and bookkeeping to stay organized.
✅ Automated Tax Savings – Easily set aside sales tax and generate reports automatically.
✅ Earn 3.00% APY*** – Grow your savings while running your business.

Already using Lili? Sync your account now for instant access to your financials while managing your store.

== Installation from within WordPress ==

1. Visit **Plugins > Add New**.
2. Search for **Lili**.
3. Install and activate the Lili plugin.

== Manual installation ==

1. Upload the entire `lili` folder to the `/wp-content/plugins/` directory.
2. Visit **Plugins**.
3. Activate the Lili plugin.

== Third Party Services ==

This plugin relies on the Lili API (https://prod.lili.co/lili/wordpress) as a core service for the following functionality:

* User authentication and two-factor verification
* Retrieving user account details and transaction history
* Managing authentication tokens and session handling

The plugin requires valid Lili API credentials (Client ID and Secret) to function. All API communications are secured and include:
* Automatic token refresh mechanism
* Retry logic for failed requests
* Request caching to optimize performance
* Error handling for API responses
